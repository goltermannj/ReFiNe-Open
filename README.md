# ReFiNe-Open
*Exploring OpenNeuro-Based Replications of Neuroimaging Findings (Brainhack OHBM 2026)*

## Background
Replication is a cornerstone of cumulative science, yet direct replications remain rare in neuroimaging despite widespread concerns regarding small sample sizes, small effect sizes, and analytic flexibility. While openly available MRI datasets provide new opportunities for replication research, identifying suitable datasets and implementing standardized replication procedures remain challenging.

## Project
ReFiNe-Open aims to explore whether openly available datasets from OpenNeuro can be systematically leveraged to conduct direct replication studies in neuroimaging.
The project builds upon the replication framework developed in the Replicability of Findings in Neuroimaging in Depression (ReFiNe-MDD) study, including predefined replication criteria, analytic workflows, and reporting standards.

## Goals for the BrainHack
1.	Assess the feasibility of applying the ReFiNe replication framework to OpenNeuro datasets.
2.	Develop tools to match published neuroimaging findings with potentially suitable OpenNeuro datasets.
3.	Identify and document practical challenges and solutions.
4.	Conduct a pilot replication attempt using one target finding and one OpenNeuro dataset.
5.	Create an open workflow that can serve as the foundation for a larger replication initiative after the BrainHack.

## Available resources
- ReFiNe replication protocol and analysis framework
- OpenNeuro datasets and annotated txt file containing dataset descriptors
- Curated list of candidate findings for replication
- Potentially deepmriprep for preprocessing (https://www.nature.com/articles/s43588-026-00953-7)

## Potential impact
The project could establish a scalable framework for conducting neuroimaging replications using openly available datasets and contribute to improving reproducibility and cumulative knowledge generation in brain imaging research.
